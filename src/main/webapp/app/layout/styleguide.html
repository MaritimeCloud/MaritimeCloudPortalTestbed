The panel element has a title-section with a title and a icon, a body-section and a footer-section with buttons. The buttons are added as 'panel-button'-elements.

The panel element has the following customizable attributes:

Mandatory:
  • title: sets the panel title
Optional:
  • category: shorthand for choosing the bootstrap contextual state class (panel-type). Defaults to "default" which yield the class panel-default. See bootstrap. Valid values are "default, primary, success, info, warning, danger"
  • title-color:
  • title-size: sets the font size of the title in pixels
  • col: shorthand of setting the class col-sm-X where X is a number from 1-12. Defaults to 6
  • offset: shorthand of col-sm-offset-X
  • icon: shorthand of setting a fontawesome icon (omit the 'fa-' prefix). Specify "none" to suppres the icon completely. Defaults to 'info-circle'
  • icon-class: Overrides the icon-class with a class of your own choise. E.g. 'glyphicon glyphicon-search'
